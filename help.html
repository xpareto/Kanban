Daily Routine

1
Open the app in the morning. If Google Calendar is connected (📅 blue in header), today's and tomorrow's events are imported automatically into NEXT with a blue background.
2
Review NEXT. Calendar events appear at the top. Tasks with ⚠ have multiple To Do items with no * priority set.
3
Process SHARE inbox — links and text received from other apps. Tick items and Merge → Clipboard to copy as a formatted list.
4
Mark tasks done with ✓. Add new tasks directly with + Next or inside a project.

NEXT Section

Priority rule — the asterisk

SituationWhat appears in NEXT
Project with 1 To Do taskAppears automatically
Project with multiple To Do tasks, one with *Only the starred ones (shown without *)
Project with multiple To Do tasks, none with *All shown with ⚠ warning

Example: project "Vacation" has 3 tasks. Write *Book flights → only that task appears in NEXT.

Reordering — multiple selection

1
Long-press (500ms) on any NEXT task → selection mode activates. The pressed row is selected (blue highlight).
2
Tap other tasks to select/deselect them.
3
Blue bar appears: ↑ Top | ↓ Bottom | ✕ Cancel
4
↑ Top: selected tasks move to the beginning, keeping their relative order. ↓ Bottom: selected tasks move to the end.
Example: NEXT = [A, B, C, D, E]. Select B and D.
→ ↑ Top: [B, D, A, C, E]
→ ↓ Bottom: [A, C, E, B, D]

Individual reorder — drag

Hold the ⠿ handle on the left of any row and drag up or down.

Inline editing

Double-tap (or double-click on desktop) on any task text → editable field. Enter or tap outside saves. Escape cancels. Edits in NEXT sync back to the source project automatically.

Promote free task to project — ⊞ button

Free tasks (not linked to a project) show a ⊞ button. Tap → bottom sheet with all projects sorted alphabetically. Tap a project to add the task there, or type a new project name.

/c Calendar Syntax

Add a calendar date directly inside any task text using the /c command. Works in NEXT and WAITING.

What you typeMeaningAfter tapping 📅
Doctor /c5.5May 5, 09:00 (default)Doctor 📅 5.may 09:00
Doctor /c5.5 14:30May 5, 14:30Doctor 📅 5.may 14:30
Pills /c1.5rz 08:00May 1, daily, 08:00Pills 📅 1.may 08:00 (zil)
Meeting /c15.5rs 10:00May 15, weekly, 10:00Meeting 📅 15.may 10:00 (sapt)
Rent /c1.6rlJune 1, monthly, 09:00Rent 📅 1.jun 09:00 (lunar)
Birthday /c15.7raJuly 15, yearly, 09:00Birthday 📅 15.jul 09:00 (anual)
Repeat codes: rz = daily · rs = weekly · rl = monthly · ra = yearly · no code = one-time
Format: /cDAY.MONTH[repeat][ HH:MM] — space before time is optional
Default time: 09:00 if no time is specified

Flow: type /c → 📅 button appears → tap 📅 → Google Calendar opens pre-filled → tap Save in Calendar → return to app. The task text updates to show the formatted date. The 📅 button stays active for changes.

Google Calendar Import

1
Tap the 📅 button in the header (grey = not connected).
2
Google login page opens → sign in and authorize ProjectBoard.
3
You return to the app. Today's and tomorrow's events are fetched automatically and appear at the top of NEXT with a blue background.
4
Events are re-fetched once per day (first open of each day).
What appears: All events today and tomorrow, except daily recurring ones (e.g. alarms).
Multi-day events: Events that started before today but haven't ended yet (e.g. "Pay utilities 1–5 Apr") appear as [ongoing 1-5 apr] Pay utilities.
Token expires after 1 hour — tap 📅 again to reconnect.
Note: The app is in "Testing" mode in Google Cloud Console. To use Google Calendar, your email must be added as a test user by the developer.

SHARE Section

Everything shared from other Android apps (browser, YouTube, WhatsApp, etc.) lands here automatically.

Merge — combine multiple items

1
Tick items with the green checkbox on the left of each row.
2
Yellow bar appears: ⧉ Merge → Clipboard | 🗑 Delete from Share
3
⧉ Merge → copies the concatenated text to clipboard (format: /c\n* item\n* item). Items stay in SHARE.
4
🗑 Delete from Share → original items are replaced by a single merged item.

Projects & Tasks

Board columns

ColumnMeaning
To Do ⚡Active tasks. 1 task → auto-appears in NEXT. Multiple → use * to declare priority.
Wait ⏳Delegated or blocked tasks. Appear in WAITING section.
Done ✅Completed tasks. Project moves to EMPTY PROJECTS when no active tasks remain.

Drag tasks between columns

Hold the task card and drag it left or right to change its column.

Reorder projects

Hold the ⠿ handle on the left of any project row and drag up or down. Same for section dividers.

Empty Projects

Projects with no active tasks appear in the EMPTY PROJECTS section. Tick the checkbox → confirm Yes to archive (removes from board, keeps in journal).

Export / Import

Access from the header: tap Data, Journal, or Clip to open the respective view with export buttons.

ButtonFull nameFile produced
EJDExport JSON with Datepb-date-2026-04-01.json — new file each time (history)
EJExport JSON fixedpb-date.json — overwrites previous (for Google Drive sync)
ETDExport Text with Datejurnal-2026-04-01.txt — readable, no diacritics, searchable
ETExport Text fixedjurnal.txt — overwrites
IImportOpens file picker for JSON (or JSON/TXT for Clipboard)
Migrate to another device:
1. On old device: Data view → EJ → save pb-date.json
2. On new device: install PWA, Data view → I → import the file
3. Reconnect Google Calendar (📅) — token doesn't migrate
Overwrite (Chrome only): First export shows "Save as" dialog — choose location once. Subsequent exports in the same session overwrite directly. On other browsers, the file downloads normally (browser may add "(1)", "(2)" suffixes).

Labels in ⚙ Settings

You can add any string as a label — not just # prefixed ones. Examples: #urgent, /c, today, @home. Tap a label in the header to instantly search for all tasks containing it.
